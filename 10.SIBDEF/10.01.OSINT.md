<a name="итоги-osint-разведки-"></a>
## Итоги OSINT - разведки:

### 1. Порт 22: OpenSSH 8.2p1 (4ubuntu0.13)

Shodan видит версию OpenSSH 8.2p1, которая в базе CVE Details ассоциируется с CVE-2024-6387, CVE-2023-48795, CVE-2025-32728. Данная конкретная сборка 4ubuntu0.13 является максимально защищенной на текущий момент для дистрибутива Ubuntu 20.04. Эксплуатация через известные CVE невозможна. Возможен только брутфорс или атака через украденные ключи. Скорее всего, имеет хорошо защищенный вход.

### 2. Порт 10050: Неизвестный сервис

По умолчанию порт закреплен за Zabbix Agent  (система мониторинга). Он используется для связи между Zabbix Server и Zabbix Agent, и не должен быть публичным. Возможна проверка отправкой запроса протокола Zabbix (заголовок ZBXD) для определения версии Zabbix Agent. CVE зависит от версии агента. Возможно, просто ловушка для отвлечения внимания.

### 3. WEB - приложения на 8050 и 7788 портах

Веб-приложения - это прикладной уровень. Их безопасность зависит от кода разработчика. Тот факт, что эти порты найдены через Google Dorking, подтверждает, что на сервере отсутствует файл robots.txt или заголовки, запрещающие индексацию, что говорит о слабой политике безопасности.

**Веб-приложения по итогам OSINT-разведки признаются приоритетными целями.**
